Вопрос: разве два ИНЬ могут дать сильную ИНЬ? Ведь ИНЬ может быть
сильной только в комплекте с ЯН, и два ИНЬ изначально слабы, как ты их ни
крути.
Чтобы прояснить, предлагаю табличку. Скажи, ты согласен с ней или нет?
Стационарные элементы - квадратные скобки,ИНЬ. Мобильные - полукруглые,
ЯН. Итак:
[ ] [ ] - слабая ИНЬ.
[ ] ( ) - сильная ИНЬ
( ) [ ] - слабый ЯН
( ) ( ) - сильный ЯН.
Все в точном соответствии с тем, что писал said.

Допетрило наконец, чего тут народ друг друга путает. Внесу и я свою
скромную лепту в этот увлекательный процесс.
Стационарные элементы - это ИНЬ, они же - СЛАБАЯ ПОЗИЦИЯ, когда
речь идет о парах.
Мобильные эл-ты - это, соотв., ЯН, они же - СИЛЬНАЯ ПОЗИЦИЯ.
Т.о., в цепочке tiger`а:
пики 9п[7п] 6п8п 10п Дп[Вп] [Кп] [Тп]
1. 9п [7п]
9п - мобила, сильная
[7п] - стационар, слабая
=>
в сочетании (см. объяснения said`а) дают СЛАБЫЙ ЯН (---)
2. 6п 8п
Оба - мобильны, оба - сильные => СИЛЬНЫЙ ЯН (---)
3. 10п - мобила, ЯН (---)
4. Дп [Вп] - аналогично 1. => СЛАБЫЙ ЯН (---)
